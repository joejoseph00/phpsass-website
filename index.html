PHPSass

What is PHPSass?

PHPSass is a PHP compiler for SASS, a popular CSS pre-processor language.

It is a fork of PHamlP that has been under active development since August 2011.

It is up-to-date with the v3.1.1.5 specification of SASS, and is tracking v3.2 development actively.

Where can I download it?

We host the code on Github: github.com/richthegeek/phpsass

Try it out

You can try the PHPSass parser online and compare it against the Ruby compiler here.
